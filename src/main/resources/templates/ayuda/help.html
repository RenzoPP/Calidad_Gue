¿Qué es esto?

Este Software es un Sistema de Facturación de uso comercial para bodegas, tiendas, minimarkets y/o pequeños establecimientos.

El Sistema cuenta con los siguientes módulos:

  • Gestión de Administradores (Administrador, Vendedor, entre otros.)
  • Gestión de Usuarios
  • Gestión de Clientes
  • Gestión de Proveedores
  • Gestión de Productos
  • Gestión de Categorías
  • Registro de Compras
  • Registro de Ventas
  • Registro de Cotizaciones
  • Kardex e Inventario
  • Boletas y Facturas de venta en formato PDF
  • Reporte de Ganancias
  • Reporte Otros
